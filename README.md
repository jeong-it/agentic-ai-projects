# ai-projects

We are currently living in a time where artificial intelligence is rapidly changing the world. AI is becoming more and more part of our daily lives.

Because of these changes, I have become very interested in AI and how it can be used to solve real problems and support creative ideas.

In this repository, I plan to build and share useful and interesting AI-related projects. These projects may focus on Generative and Agentic AI or explore different areas such as automation and data analysis.

My main goal is not only to write code directly, but also to learn how to use AI effectively to achieve specific goals. In other words, I want to explore how humans and AI can work together as builders.

I will mainly use Codex, but depending on the project, I may also use other AI tools such as Claude Code, GitHub Copilot, or Cursor.

This repository is a place to explore how far we can go by working together with AI.
