# Memory Game

A simple browser-based memory card game created with Codex CLI assistance.

## Updates & Enhancements
- **Fixes**: Resolved an issue where card images were not rendering in the browser when running the original `index.html` generated by Codex CLI. The code has been successfully debugged and updated using Gemini.
- **Future Ideas**: Potential improvements include adding a game timer, adjustable difficulty settings based on levels, and customizable card themes/images.

## Features
- Card matching system
- Shuffle logic
- Score tracking
- Responsive UI

## Tech Stack
- HTML
- CSS
- JavaScript
- Codex CLI
- Gemini (for debugging)
